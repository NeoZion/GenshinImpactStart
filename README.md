# GenshinImpactStart

本项目检测白屏自动播放原神启动动画和音乐，播放完毕程序自动退出。

#### 介绍

##### 支持列表

- [x] 后台运行
- [x] 自动退出

#### 如何使用

```
解压GenshinImpactStart.zip 直接运行GenshinImpactStart.exe即可
```
